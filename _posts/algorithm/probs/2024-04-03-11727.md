### [문제](https://www.acmicpc.net/problem/11727)

쉬운 문젠데 한참 헤멨다

헷갈릴 땐 점화식을 구해보는 게 좋을까

`f(n) = f(n - 1) + f(n - 2) * 2`

`f(n - 2) * 2`를 하는 이유는? `2 X 2` 공간을 채우기 위해선 

- (2 x 1) x 2 
- (1 x 2) x 2
- (2 x 2)

세가지가 가능한데, 두 번째 경우는 `f(n - 1)`과 겹치니까! 

두가지 경우의 수가 나온다

---
{: data-content=" 코드 "}

```c++
#include <bits/stdc++.h>

using namespace std;

int dp[1003];

int main() {
	int n;

	cin >> n;

	dp[1] = 1; dp[2] = 3;	

	for (int i = 3; i <= n; i++)
		dp[i] = (dp[i - 1] + dp[i - 2] * 2) % 10007;

	cout << dp[n] << "\n";
}
```
